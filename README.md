
# envctl

## Problem

Storing and managing environment variables is a messy task.

## Download

### Via Golang

```bash
go install github.com/Siddhesh-Agarwal/envctl
```

### Via Source

To build and install the tool:

```bash
# Build
go build -o envctl ./cmd/envctl

# Install system-wide
go install ./cmd/envctl
```
